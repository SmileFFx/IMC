# Calculadora de IMC

Calculadora de IMC (Índice de Massa Corporal) em Python que solicita nome, idade, altura e peso do usuário, calcula o IMC e classifica em categorias como abaixo do peso, peso ideal, sobrepeso ou obesidade. Exibe mensagens personalizadas e permite reiniciar ou encerrar o programa.

---

## Funcionalidades

- Solicita ao usuário:
  - Nome e idade
  - Altura (em metros) e peso (em kg)
- Calcula o IMC:
```

IMC = peso / (altura ** 2)

````
- Classifica o IMC em categorias:
- Abaixo do peso (≤ 18.5)
- Peso ideal (18.5 – 24.9)
- Sobrepeso (25 – 29.9)
- Obesidade (≥ 30)
- Permite reiniciar o programa ou encerrar ao final.
- Trata entradas inválidas e aceita vírgula ou ponto como separador decimal.

---

## Como usar

1. Clone ou baixe o repositório:
 ```bash
 git clone https://github.com/SmileFFx/IMC.git
 cd IMC
````

2. Tenha Python 3 instalado.
3. Execute o script:

   ```bash
   python IMC.py
   ```
4. Siga as instruções no terminal.
